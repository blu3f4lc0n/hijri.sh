hijri.sh
========

Simple bash script to display Islamic date (Hijri / Hijriyah). Ported from php script I found online. I do not own the calculation method.

####USAGE
Make executable and run:
```bash
$ chmod +x hijri.sh
$ ./hijri.sh
```

Output:
```bash
13 Rabiul Awwal 1435 H
```
